## 📌 Featured Projects  

### 🚗 Vehicle Detection and Counting  
A web app for automated vehicle detection and counting. Built with **Python, Flask, and OpenCV**, it processes uploaded images to identify and classify cars and buses, providing a user-friendly and scalable solution for **traffic analysis and urban planning**.  

---

### 🧬 Breast Cancer KNN Regression  
Regression model using **KNN in Python**. Includes data preprocessing, statistical analysis, and feature visualization. Achieved **93.9% accuracy** with balanced precision (0.97 Benign, 0.89 Malignant) using 20% test split for robust evaluation.  

---

### 🛒 Online Auction System  
Java-based **online auction system** showcasing OOP concepts (inheritance, encapsulation, polymorphism). Simulates a virtual auction environment where sellers list items and bidders compete, managing data, bids, and winners efficiently.  

---

### 🖥️ MLFQ CPU Scheduling  
Implemented **Multi-Level Feedback Queue (MLFQ)** scheduling in Python with **Tkinter GUI**. Users can input process details and visualize process movement across queues, modeling **real-world scheduling behavior** with preemption and starvation prevention.  

---

### 🍴 TastyEats – Interactive Recipe Website  
An engaging recipe platform with **user authentication, dynamic search, and visual recipe gallery**. Built for food enthusiasts to explore global cuisines with seamless navigation and an intuitive design.
